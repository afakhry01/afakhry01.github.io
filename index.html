ePaper Tag

An ePaper tag which is connected to an old raspberry pi 2 B through wifi. It shows the name in red, status (like out for lunch, in a meeting, ..etc) and today's quote in black.

Release Date: July 2019

Machine Vision Robot

A preliminary model for a robot that can identify faces and spell out names. A beaglebone black board will be used for this project with Yocto and openCV (in C).

Release Date: December 2019

Pixel Drawing Panel

A pixel drawing panel that you can use to do nice drawings and save them as pictures. You may also change the drawing panel size and use different colors. It was built using HTML, CSS and jQuery.

Piano Player

A piano player website which allows basic music playing for ear training and some basic composing. Use keyboard buttons to play the music scale. It was built using HTML, CSS and vanilla javascript.
